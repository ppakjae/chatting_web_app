## 설치 방법
#### (node.js 와 npm이 설치되어 있다는 전제)
~~~
git clone https://github.com/zzangjames/chatting_web_app
cd chatting_web_app //(클론한 해당 경로로 이동)
npm i 
npm run build
cd Owl-darwin-x64 //(build된 경로로 이동)
./Owl
~~~
